Cruncher provides three percentage tools that each solve a different variant of percentage arithmetic. All three are available in the standard tier (the default). Using dedicated tools rather than asking your AI to compute percentages inline eliminates the risk of hallucinated results — large language models frequently produce small but critical rounding errors on percentage math.

<Tip>
  Even simple percentage calculations benefit from using Cruncher. AI models can hallucinate on percentage math, especially when numbers involve decimals or multi-step reasoning. These tools return exact results every time.
</Tip>

---

## percentage_of

Answers the question: **"What is X% of Y?"**

Given a percentage and a total, returns the portion that percentage represents.

<ParamField body="percent" type="number" required>
  The percentage value (e.g., `15` for 15%). Do not include the `%` symbol — pass the numeric value only.
</ParamField>

<ParamField body="total" type="number" required>
  The total or whole from which the percentage is taken.
</ParamField>

**Formula:** `result = (percent / 100) × total`

```json example
{
  "tool": "percentage_of",
  "arguments": { "percent": 15, "total": 200 }
}
```

**Result:** `30`

15% of 200 is 30.

## percentage_change

Answers the question: **"Going from A to B, what is the percentage change?"**

Returns the percentage increase or decrease between two values. A positive result indicates growth; a negative result indicates a decrease.

<ParamField body="from" type="number" required>
  The starting value (original).
</ParamField>

<ParamField body="to" type="number" required>
  The ending value (new).
</ParamField>

**Formula:** `result = ((to − from) / |from|) × 100`

```json example
{
  "tool": "percentage_change",
  "arguments": { "from": 50, "to": 80 }
}
```

**Result:** `60`

50 → 80 is a 60% increase.

<CodeGroup>

```json price increase
{
  "tool": "percentage_change",
  "arguments": { "from": 120, "to": 150 }
}
```

```json stock decline

  "arguments": { "from": 340, "to": 289 }
}
```

```json growth rate
{
  "tool": "percentage_change",
  "arguments": { "from": 1000, "to": 1250 }
}
```

</CodeGroup>

## percentage_reverse

Answers the question: **"X is Y% of what total?"**

Given a value and the percentage it represents, returns the original total.

<ParamField body="value" type="number" required>
  The known partial value (the portion).
</ParamField>

<ParamField body="percent" type="number" required>
  The percentage that the value represents (e.g., `15` for 15%).
</ParamField>

**Formula:** `result = (value / percent) × 100`

```json example
{
  "tool": "percentage_reverse",
  "arguments": { "value": 30, "percent": 15 }
}
```

**Result:** `200`

30 is 15% of 200.

```json find the original price
{
```

```json find the survey population
}
```

```json find a total from a tax amount
{
  "tool": "percentage_reverse",
  "arguments": { "value": 102, "percent": 8.5 }
}
```

</CodeGroup>
