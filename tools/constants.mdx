Cruncher provides 16 named constants through the `get_constant` tool, spanning mathematics, physics, and chemistry. The tool is available in the standard tier (the default). Constants can also be used directly inside `evaluate_expression` strings, which lets you combine them with arithmetic, trigonometry, and other operations in a single call.

---

## get_constant

Returns the exact numeric value of the named constant.

<ParamField body="name" type="string" required>
  The constant identifier. Must be one of the 16 names listed below:

  `"pi"`, `"e"`, `"tau"`, `"phi"`, `"sqrt2"`, `"euler_mascheroni"`, `"c"`, `"g"`, `"G"`, `"h"`, `"k"`, `"R"`, `"NA"`, `"e_charge"`, `"m_e"`, `"m_p"`
</ParamField>

### Examples

<CodeGroup>

```json pi
{
  "tool": "get_constant",
  "arguments": { "name": "pi" }
}
```

```json speed of light
{
  "tool": "get_constant",
  "arguments": { "name": "c" }
}
```

```json Avogadro's number
{
  "tool": "get_constant",
  "arguments": { "name": "NA" }
}
```

```json golden ratio
{
  "tool": "get_constant",
  "arguments": { "name": "phi" }
}
```

</CodeGroup>

## Mathematical constants

| Name | Description | Exact value |
|------|-------------|-------------|
| `pi` | π — ratio of a circle's circumference to its diameter | 3.14159265358979... |
| `e` | Euler's number — base of the natural logarithm | 2.71828182845904... |
| `tau` | τ = 2π — full turn in radians | 6.28318530717958... |
| `phi` | φ — golden ratio, (1 + √5) / 2 | 1.61803398874989... |
| `sqrt2` | √2 — square root of two | 1.41421356237309... |
| `euler_mascheroni` | γ — Euler-Mascheroni constant | 0.57721566490153... |

## Using constants in evaluate_expression

All 16 constant names are recognized inside `evaluate_expression` strings. This lets you perform multi-step calculations in a single tool call without first looking up a constant's value.

```json circumference of a circle with radius 5
{
  "tool": "evaluate_expression",
  "arguments": { "expression": "2 * pi * 5" }
}
```

**Result:** `31.41592653589793`

<CodeGroup>

```json energy using E = mc²
{
  "tool": "evaluate_expression",
  "arguments": { "expression": "1 * c ^ 2" }
}
```

```json golden ratio squared
{
  "tool": "evaluate_expression",
  "arguments": { "expression": "phi ^ 2" }
}
```

```json gravitational constant scaled
{
  "tool": "evaluate_expression",
  "arguments": { "expression": "G * 1e11" }
}
```

```json √2 times itself
{
  "tool": "evaluate_expression",
  "arguments": { "expression": "sqrt2 * sqrt2" }
}
```

```json e squared
{
  "tool": "evaluate_expression",
  "arguments": { "expression": "e * e" }
}
```

</CodeGroup>

<Note>
  Use explicit multiplication: `2 * pi` is valid, but `2pi` is not. Cruncher does not support implicit multiplication. Constants and scientific notation coexist safely — `1e6` and `e * 2` both work correctly without ambiguity.
</Note>
