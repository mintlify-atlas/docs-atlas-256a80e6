Cruncher MCP runs as a local server that your AI client connects to over stdio. This quickstart walks you through adding Cruncher to Claude Desktop — the most common setup — and verifying it works with a few example calculations. The entire process takes under 5 minutes and requires no compilation or package installation beyond Node.js.

## Prerequisites

You need Node.js 18 or newer installed on your system. Verify your version:

```bash
node --version
```

If you see `v18.0.0` or higher, you're ready. Download Node.js from [nodejs.org](https://nodejs.org) if needed.

## Add Cruncher to Claude Desktop

<Steps>
  <Step title="Locate your Claude Desktop config file">
    Find the configuration file for your operating system:

    | OS | Path |
    |----|------|
    | macOS | `~/Library/Application Support/Claude/claude_desktop_config.json` |
    | Windows | `%APPDATA%\Claude\claude_desktop_config.json` |
    | Linux | `~/.config/Claude/claude_desktop_config.json` |

    Create the file if it doesn't exist yet.
  </Step>
  <Step title="Add the Cruncher server configuration">
    Add the following JSON to your config file. If you already have other MCP servers configured, add the `cruncher` entry inside the existing `mcpServers` object:

    ```json claude_desktop_config.json
    {
      "mcpServers": {
        "cruncher": {
          "command": "npx",
          "args": ["@slbdn/cruncher-mcp"]
        }
      }
    }
    ```

    This uses `npx` to download and run the latest version of Cruncher automatically. No separate installation step is needed.

    <Tip>
      `npx` caches downloaded packages locally, so after the first run, subsequent starts are fast.
    </Tip>
  </Step>
  <Step title="Restart Claude Desktop completely">
    Quit the Claude Desktop app entirely — don't just close the window — and relaunch it. Claude will connect to Cruncher on startup and register all 34 tools from the default `standard` tier.
  </Step>
  <Step title="Verify the connection">
    Open a new conversation in Claude Desktop and ask a math question that requires a tool:

    > "What is 15% of 240?"

    You should see Claude call the `percentage_of` tool and return `36` as the result. If the tool call appears in the interface, Cruncher is connected and working.
  </Step>
</Steps>

## Try example calculations

Once Cruncher is connected, ask your AI assistant any of these questions to explore different tool categories:

**Arithmetic and expressions**
> "Evaluate the expression: sin(pi/4) + sqrt(16) + log10(100)"

**Statistics**
> "Calculate the average, median, and standard deviation of [15, 22, 8, 41, 19, 30]"

**Unit conversion**
> "Convert 26.2 miles to kilometers"

**Physical constants**
> "What is the speed of light in meters per second?"

**Percentages**
> "A stock went from $50 to $75. What is the percentage change?"

**Full tier (requires `CRUNCHER_TOOL_SET=full`)**
> "Store 99 in memory, add 5, then recall the total"
